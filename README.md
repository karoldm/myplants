### Getting Started

1. Install [Flutter](https://flutter.dev)

2. Clone the repo

```sh

git clone https://github.com/karoldm/myplants.git

```

3. Get dependencies

```sh

flutter pub get

```

3. Run app

```sh

flutter run

```
